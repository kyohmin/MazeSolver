# Maze Generator & Solver
**The program consists three components:**
  1. Maze Generator
     - GIF, PNG, and Text Generator
  2. Maze Solver
     - Depth First Search (DFS) method
  3. Progress Visualizer
     - Pygame sequence vs parallel
